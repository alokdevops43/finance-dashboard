# 💰 Expense Tracker Dashboard

A modern **SaaS-inspired Expense Tracker Dashboard** built using **Vanilla HTML, CSS, and JavaScript**.

Designed with a premium **glassmorphism interface**, responsive layout, smooth animations, and Local Storage support, this project helps users manage their personal finances efficiently while showcasing advanced frontend development skills.

---
##Live link

https://alokdevops43.github.io/finance-dashboard/

- 📊 Interactive Dashboard
- 💰 Total Balance
